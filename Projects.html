Spam Classifier

The SMS Spam Classifier is a machine learning-based system that detects and filters spam messages. It analyzes text patterns using NLP techniques to differentiate between spam and legitimate messages, ensuring a clutter-free inbox.

Book recommender System

The Book Recommender System suggests books based on user preferences and reading history. It leverages machine learning and collaborative filtering techniques to provide personalized recommendations, enhancing the reading experience.

Disease Prediction

The Disease Prediction System analyzes user symptoms using machine learning to predict potential diseases, helping in early diagnosis and timely medical consultation.

Gesture Control Game

The Need for Speed Gesture Control Game uses computer vision to detect hand gestures for controlling in-game actions like steering, acceleration, braking, and nitro boost. This project enhances the gaming experience by replacing traditional controls with intuitive hand movements.
